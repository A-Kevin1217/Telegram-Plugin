## 安装教程

1. 准备：[TRSS-Yunzai](../../../Yunzai)
2. 输入：`#安装Telegram-Plugin`
3. 打开：[BotFather](https://t.me/BotFather) 创建 Bot：  
① /newbot  
② /mybots → API Token → 得到 `Token`  
4. 输入：`#TG设置Token`

## 使用教程

- #TG账号
- #TG设置 + `Token`
- #TG代理/反代 + `scheme://[userinfo@]host[:port]`
